API: Application Programming Interface
     HTTP kérésre adatbázisműveleteket végzek, szituációtól függően ha szükséges adatokat küldök vissza JSON formátumban
     Használata: res.json()

SSR: Server Side Rendering
     HTTP kérésre megjelenítendő HTML kódot küldök vissza
     Használata: res.render()

Beüzemelés:
npm install
node app.js
